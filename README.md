# myprojectedrpg
Bienvenue dans MyProjectRPG !

Oui bon le nom est pas ouf mais je n'ai pas spécialement de nom pour ce jeu (pas encore?).

En fait ce n'est pas exactement un jeu, mais un concept de jeu que j'ai imaginé il y a quelques temps et que je souhaite expérimenter, voir si les mécaniques fonctionnent.

Voici pourquoi ce que vous voyez est un pauvre petit programme en python.
Bref en espérant que le concept vous parle !

----------

# Manuel de jeu
Le but de ce jeu tourne autour de la mécanique de fusion.
Vous pouvez acheter des techniques de combat, des armes, des formes de magie et des objets. Ensuite libre à vous de choisir comment fusionner ces éléments.

Exemple : fusionner la compétence "Coup simple" et "Magie élémentaire de feu" vous donne la compétence "Boule de feu simple".

Une fois paré au combat vous n'avez plus qu'a cliquer sur "Partir à l'attaque", ce qui déclenchera une phase de combat contre 5 ennemis aléatoire.
(Ceci n'est qu'un exemple a terme j'aimerais vous proposer des quêtes que vous pourrez résoudre avec vos éléments)
